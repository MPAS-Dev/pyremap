pyremap.descriptor.LatLon2DGridDescriptor¶

class pyremap.descriptor.LatLon2DGridDescriptor[source]¶

A class for describing a lat-lon grid that may not be a tensor grid (lat/lon are 2D arrays). The grid is assumed to be regional, since this is difficult to determine just from the lat/lon values. The calling code should set regional = False for global grids with 2D lat/lon
