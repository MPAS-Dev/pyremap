pyremap.descriptor.PointCollectionDescriptor.to_scrip

PointCollectionDescriptor.to_scrip(scripFileName, expandDist=None, expandFactor=None)[source]

Given an MPAS mesh file, create a SCRIP file based on the mesh.

Parameters:

  • scripFileName (str) – The path to which the SCRIP file should be written

  • expandDist (float or numpy.ndarray, optional) – A distance in meters to expand each grid cell outward from the center. If a numpy.ndarray, one value per cell.

  • expandFactor (float or numpy.ndarray, optional) – A factor by which to expand each grid cell outward from the center. If a numpy.ndarray, one value per cell.
